hexo修改博客主题

发表于 2020-09-27 分类于 hexo ， next

上期回顾

上一期我们讲了如何用hexo搭建博客,本期我们来讲一下如何修改博客的主题~

  • 博客刚刚搭建出来,会有一套默认的系统主题
  • 但是其实博客的主题真的是千变万化, 多种多样, 色彩缤纷, 五光十色…

本期内容

挑选主题

  • hexo为我们提供的主题有很多, 我们可以根据自己的喜好去选择, 查看更多主题
  • 我选择的是next

下载主题

  • 进入到自己博客项目 git clone https://github.com/theme-next/hexo-theme-next themes/next

  • 这时你在themes目录下就可以看到了

修改配置文件

  • 在博客项目根目录到_config.yml搜索theme, 当前默认主题应该是landscape, 我们改成next
  • 这个时候我们输入命令hexo server启动, 就可以在本地看到新的主题http://localhost:4000

写博客时, 可使用<!--more-->来实现阅读更多

设置canvas

什么是canvas ? 就是我们经常看到的, 别人的博客页面, 经常会有一些线条,可以根据鼠标汇聚的那种动画!

下载canvas-nest
  • 进入到next目录下
  • 将canvas-nest下载到next目录下的source/lib目录下
  • 配置insulin配置文件_config.yml中的canvas_nest(如果没有该配置, 自行添加)

重新启动后, 效果就来了!

还有很多其他的特效等待着你们去挖掘!

设置看板娘

作为一个软妹子/硬汉, 怎么能没有看板娘呢! 更多看板娘

注意: 以下npm如果网络不通, 可使用cnpm

  • 在下载前, 我们先看一下根目录下的package.json, 是否已经有hexo-helper-live2d依赖, 如果有,先进行卸载
  • 下载hexo看板娘支持, 在根目录执行
  • 下载自己选择的看板娘模型, 我选的是我家猫咪同款的live2d-widget-model-tororo
  • 修改根目录_config.yml, 添加配置, use使用自己选择的模型包名
  • 本地重启, 就可以看到啦~

最后我们再将代码重新部署到github就欧克了!
